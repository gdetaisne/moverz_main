Incluse base : Franchise 800-1 500€ (casse = vous payez jusqu'à 1 500€). Tous risques +10-15€/j : Franchise 0-300€. Recommandé si inexpérience conduite utilitaire, centre Strasbourg dense (rues étroites, tram, piétons). Carte bancaire Gold/Platinum couvre parfois (vérifier).

## Assurance Base Incluse

**Toujours incluse** : Location utilitaire = assurance base obligatoire.

**Couverture** :  
- Dommages tiers (voitures heurtées, piétons, immeubles)  
- Vol véhicule  
- Incendie  
- Bris de glace

**Franchise** : 800-1 500€ selon agence, taille véhicule.

**Conséquence** : Rayure portière 500€ = vous payez 500€. Rétro cassé 1 200€ = vous payez 1 200€ (franchise 1 500€ non atteinte).

## Assurance Tous Risques

**Optionnelle** : +10-15€/jour.

**Franchise réduite** : 0-300€ (vs 800-1 500€ base).

**Exemple** : Rétro cassé 1 200€. Tous risques franchise 300€ = vous payez 300€ (vs 1 200€ base). Économie 900€.

**Coût** : Master 60€/j + tous risques 15€ = 75€/j. Weekend 150€ → 180€ (+30€).

## Quand Souscrire ?

**Recommandé si** :  
- **Inexpérience** : Première location utilitaire (gabarit 2m vs voiture 1,7m).  
- **Centre Strasbourg** : Grande Île (rues 2-4m, tram, piétons nombreux = risques accrochage).  
- **Objets valeur** : Transport meubles 5 000€+ (stress conduite = risque accident).  
- **Étages multiples** : Fatigue portage = concentration réduite conduite retour.

**Pas nécessaire si** :  
- Expérience utilitaires (plusieurs déménagements)  
- Quartiers faciles (Neudorf, Robertsau rues larges)  
- Budget très serré (risque acceptable)

## Carte Bancaire

**Gold/Platinum** : Certaines couvrent franchise location véhicule (vérifier contrat carte).

**Visa Premier** : Franchise max 900€ souvent (vs 1 500€ agence = économie partielle).

**Mastercard Gold** : Idem 900-1 200€.

**Conditions** :  
- Payer location carte (débit complet)  
- Refuser assurance agence (sinon carte n'intervient pas)  
- Déclarer sinistre assurance carte sous 20j

**Vérification** : Appeler assurance carte AVANT location (confirmer couverture utilitaires, franchises, exclusions).

## Exclusions

**Assurance ne couvre PAS** :  
- Conduite sous alcool (0,5g/L dépassé = nullité)  
- Conducteur non déclaré (ami conduit sans être sur contrat)  
- Usage pro si location particulier  
- Négligence grave (clés laissées, portières ouvertes)

## Dommages Fréquents

**Rétros** : Cassés ruelles étroites (300-800€ réparation).

**Rayures** : Portières, hayon (200-500€).

**Bris de glace** : Pare-brise fissuré (500-1 200€). Souvent couvert base.

**Chocs arrière** : Marche arrière difficile utilitaire (300-1 000€).

## Recommandation

**Première location** : Tous risques (sérénité 15€ vs risque 1 500€).

**Habitué** : Base suffit (économie 30-45€ weekend).

**Grande Île** : Tous risques (rues étroites = risque élevé).

**Carte Gold** : Vérifier couverture (alternative gratuite tous risques).
